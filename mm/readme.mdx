# This guide introduces a simple market maker bot for Drift that uses oracle-based limit orders to peg its quotes to an external price feed (PYTH).

## Setting up the bot
Dependencies
To run the bot, you will need:
- Node.js and Typescript
- Drift SDK @drift-labs/sdk
- Solana Web3.js, v1.92.3
- (Optional) dotevn, for managing environment variables
